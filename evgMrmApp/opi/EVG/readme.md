## Quick start
run 

    ./startUI.sh -s <system name>
from this folder.


Options:

- -s <system name\> ..... The system/project name
- -d <EVG name\>    ..... Event Generator / timing card name (default: EVG0) 
- -f <form factor\> ..... Choose the event generator form factor (default: VME-300)
 
	                     Choices: VME, VME-300

 - -h ..... This help
